# Glass Shadow

A tactical espionage PWA game built with Next.js and React.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deploy to Vercel

1. Push to GitHub
2. Import repo at vercel.com
3. Deploy

## Features

- Silver/Chrome UI palette with optional DOS green mode
- Sloane AI handler with conversation system
- Fog-of-war tactical map
- Command line interface (LOOK, INSPECT, TALK, ACT)
- Consequential vitals system (pulse affects performance)
- Multiple challenge types: Human, Terminal, Search, Surveillance, Puzzle

## PWA

Add to home screen on iOS/Android for full-screen native-like experience.
